### Sparrow is small, but it is all-encompassing
Support CSS Modules <br/>
Support Import Static File <br/>
Support Less <br/>
Support Decorator <br/>
Not Support SSR! <br/>

#### Start
```
   npm start
```

#### Build
```
   npm run build
```
